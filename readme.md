# Hi, welcome!

This project was designed for determining the time and distance between two stops on tram networks in Melbourne, Australia.

Tram time tables are available as PDFs, which are then converted to HTML using PDFMiner and scraped using Beautiful Soup.




### Requirements
-[PDFMiner](https://github.com/euske/pdfminer)
-[Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/)
